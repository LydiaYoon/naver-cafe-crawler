[네이버 카페 게시물 댓글수, 조회수 크롤러]
최초 제작 : 2019년 9월 6일
마지막 수정 : 2020년 6월 18일

1. 실행 시 필요한 프로그램 및 라이브러리
- Eclipse, pyDev
- Python 3.8.3
- ChromeDriver 83.0.4103.39
- openpyxl 3.0.3
- selenium 3.141.0
- BeautifulSoup4 4.9.1

* openpyxl, selenium, BS4는 pip를 통해 간단하게 설치 가능합니다!
* ChromeDriver는 자신의 PC에 설치된 크롬 버전에 맞게 설치해주세요.


2. 실행 방법
1) crawler.py 파일을 이클립스로 열어주세요.
2) 크롬 드라이버를 driver폴더 밑에 설치해주세요.
3) 입력 파일의 형식은 test.xlsx를 참고해주세요. 열 위치가 변동될 경우 올바르게 작동하지 않습니다.
4) 입출력 파일명은 확장자 없이 입력해주세요.
5) 콘솔창에 '출력 파일 저장 완료'가 보이면 실행 완료입니다.


3. 주의사항
- Windows 환경에서 실행되도록 수정했습니다. (2020-06-18)
- 크롬 드라이버와 PC에 설치된 크롬 브라우저간 버전이 일치하지 않을 경우 실행이 되지 않습니다.
  크롬 버전은 chrome://chrome 에서 확인 가능합니다.
- 타임 슬립과 로딩 대기시간을 삭제하면 에러가 발생할 수 있습니다.
- 추후 네이버 정책 상 selenium 같은 자동화 도구 등 일반적이지 않은 방법으로 로그인 시
  자동입력방지 코드 입력을 해야할 수도 있습니다.
  이 경우 로그인 부분 로직의 변경이 필요합니다
- 게시물이 삭제되어있는 경우 조회수와 댓글수 대신 '삭제된 게시물'이라는 문자열이 입력됩니다.
- 간혹 게시물 페이지에서 조회수를 찾을 수 없는 경우가 종종 있습니다.
  이 경우 조회수 열에 '조회수 찾기 불가'로 입력되게 했습니다.
- 확인이 쉽도록 출력파일명에는 자동으로 년월일, 시분초 값이 붙습니다.
- 네이버 로그인시 필요한 사용자 아이디와 비밀번호 정보는 수집하지 않습니다.
