[네이버 카페 게시물 댓글수, 조회수 크롤러]
2019년 9월 6일


1. 실행 시 필요한 프로그램 및 라이브러리
- Eclipse, pyDev
- Python 3.7
- ChromeDriver 76.0.3809
- openpyxl 2.6.3
- selenium 3.141.0
- BeautifulSoup4 4.8.0

* openpyxl, selenium, BS4는 pip를 통해 간단하게 설치 가능합니다!
* ChromeDriver는 자신의 PC에 설치된 크롬 버전에 맞게 설치해주세요.


2. 실행 방법
1) taehun.py 파일을 이클립스로 열어주세요.
2) 크롬 드라이버를 설치한 경로를 수정해주세요. (12 Line)
3) 네이버 로그인시 필요한 사용자 아이디와 비밀번호 라인을 수정해주세요. (21~22 Line)
4) 크롤링할 네이버 카페 게시물의 링크가 들어있는 엑셀 파일 경로를 수정해주세요. (31 Line)
5) 결과가 저장될 파일이 위치할 경로를 수정해주세요. (97 Line)
6) 변경사항 저장 후 실행합니다. 콘솔창에 'OK'가 보이면 실행 완료입니다.


3. 주의사항
- MacOS 환경에서 만들었습니다. Windows 환경에서 간혹 충돌 오류가 발생할 수 있습니다.
- 크롬 드라이버와 PC에 설치된 크롬 브라우저간 버전이 일치하지 않을 경우 실행이 되지 않습니다.
  크롬 버전은 chrome://chrome 에서 확인 가능합니다.
- 타임 슬립과 로딩 대기시간을 삭제하면 에러가 발생할 수 있습니다.
- 추후 네이버 정책 상 selenium 같은 자동화 도구 등 일반적이지 않은 방법으로 로그인 시
  자동입력방지 코드 입력을 해야할 수도 있습니다.
  이 경우 로그인 부분 로직의 변경이 필요합니다.
- 읽어들일 엑셀 파일에서 URL 열의 값이 반드시 존재한다는 가정하에 만들어졌습니다.
  URL 열에 빈 값 (공백)이 있어서는 안됩니다.
- 게시물이 삭제되어있는 경우 조회수와 댓글수 대신 '삭제된 게시물'이라는 문자열이 입력됩니다.
- 간혹 게시물 페이지에서 조회수를 찾을 수 없는 경우가 종종 있습니다.
  이 경우 조회수 열에 '조회수 찾기 불가'로 입력되게 했습니다.
- 에러가 발생할 수 있기 때문에 입력 파일명과 출력 파일명을 다르기 지정하는 것을 추천합니다.
